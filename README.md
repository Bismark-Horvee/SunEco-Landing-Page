# SunEco-Landing-Page


A modern, responsive landing page for SunEco - a sustainable energy solutions company.


## Features
- Responsive navigation bar with hero section
- Call-to-action buttons ("Get Started" and "Explore More")
- Font Awesome icons integration
- Clean semantic HTML structure

## Files
- `index.html` - Main HTML file
- `index.css` - Styling (referenced but not included)


## Installation
1. Clone or download this repository
2. Ensure `index.css` is in the same directory as `index.html`
3. Open `index.html` in your web browser
